# Feature-Aligned Speech Watermarking for Robustness to Reconstruction Distortions

Anonymous official demo for "Feature-Aligned Speech Watermarking for Robustness to Reconstruction Distortions" (ICME 2026).

This repository is released for double-blind review.

## Demo

Visit the [demo](https://anonymouswatermark.github.io/icme2026_wm/) to compare different audio watermarking methods.

## Features

- Compare 6 watermarking methods side-by-side
- Audio samples from LibriSpeech, LJSpeech, and VCTK datasets
